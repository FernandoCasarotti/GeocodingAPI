## Requirements

```
PostgreSQL >= 9.5.8
PostGIS >= 2.3
NodeJS >= 4.2.6
```


## Running the API

```
$ git clone https://github.com/Pauliceia/GeocodingAPI.git
$ cd GeocodingAPI/

$ npm install
$ npm start
```

## Running the tests

```
$ npm test
```
